# GymPlanner

A web-based gym workout planner to help users create, track, and visualize their fitness routines.

## Structure
- `index.html`: Main HTML file
- `styles/`: CSS files
- `scripts/`: JavaScript files
- `assets/`: Fonts, icons, and other static assets
- `images/`: App images and illustrations
- `data/`: Optional data files (e.g., sample workouts)

## Features
- Dashboard with stats
- Pre-made and custom workouts
- Workout execution and tracking
- History and progress visualization
- Achievements and milestones
